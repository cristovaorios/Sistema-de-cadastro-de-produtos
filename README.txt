Sistema de Estoque - Aluno: Cristóvão Carvalho

Contexto: 

O sistema desenvolvido fica responsável por lidar com o estoque de um pequeno mercado, cadastrando nome, descrição, unidade, datas de validade e fabricação, e outras informações adicionais sobre determinados produtos.

Aplicação:

Ao executar a aplicação, você irá se deparar com uma tela de login que irá pedir usuário e senha. Para acessar a tela de cadastro de produtos, digite o usuário (admin) e a senha (123).

OBS: 

No projeto, foi decidido utilizar uma fonte não presente no Visual Studio (Fonte Poppins). Para realizar uma boa execução, siga o caminho: Estoque -> Mercado-Av01 ->   Resources, clique duas vezes no arquivo Poppins-Regular.ttf e instale a fonte.

O projeto foi desenvolvido em .Net Framework.
